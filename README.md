
# Hello app by PyQt5

Testing a new app


## Run Locally

Clone the project

```bash
  git clone https://github.com/atabekdemurtaza/App_GUI
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
